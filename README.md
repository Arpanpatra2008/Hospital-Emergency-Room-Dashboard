# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze the performance of a Hospital Emergency Room (ER). It helps track patient flow, waiting time, satisfaction, admissions, referrals, and other important healthcare metrics.

---

## 🎯 Objectives
- Monitor Emergency Room performance.
- Analyze patient waiting time.
- Track patient satisfaction.
- Identify admission and referral trends.
- Support data-driven decision making.

---

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- SQL
- DAX
- Power Query

---

## 📊 Dashboard KPIs
- Total Patients
- Average Wait Time
- Patient Satisfaction Score
- Patients Referred

---

## 📈 Dashboard Features
- Patient Admission Status
- Patients by Age Group
- Patients by Gender
- Patients by Race
- Department Referrals
- Patients Seen Within 30 Minutes
- Patients by Day and Hour (Heatmap)
- Monthly Filters (Year & Month)
